================================
How to use 5-text_indentation.py
================================

``text_indentation(...)`` prints a text with 2 new lines after each of these characters: [".","?", ":"].

    >>> text_indentation = __import__("5-text_indentation").text_indentation

text:
    is arguments ``text_indentation(...)`` recieves.

Usage
=====

``text`` must be a string.
    >>> text_indentation("this is geda siraj. mohammed? alx software engineer: lives in USA") #doctest: +NORMALIZE_WHITESPACE
    this is geda siraj.
    <BLANKLINE>
    mohammed?
    <BLANKLINE>
    alx software engineer:
    <BLANKLINE>
    lives in USA 

    >>> text_indentation(5)

    TypeError: text must be a string
